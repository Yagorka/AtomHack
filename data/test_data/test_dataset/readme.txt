
Тестовый датасет представляет из себя файл в расширении .xlsx - список проиндексированных вопросов.
Ваш сабмит должен представлять из себя тот же файл (submission_<название команды>.xlsx) с заполненными колонками 'Ответ', 'Имя файла с инструкцией', 'Раздел в инструкции', 'Страница'.
Если ваша модель не выдала ответ на заданный вопрос - в графе ответ должно быть внесено сообщение вида "Простите, у меня недостаточно информации." (Содержание сообщения на ваше усмотрение).
Если модель опирается на данные, которые невозможно проиндексировать (невозможно заполнить поля 'Имя файла с инструкцией', 'Раздел в инструкции', 'Страница') - просто не заполняйте эти поля!
