В поле «Вид бюджета»  следует выбрать, для какого типа бюджета будет 
применяться создаваемая настройка Операционного пла на. После выбора вида бюджета 
соответствующим образом измениться содержимое вклад ки «Аналитики» : в зависимости 
от структуры оперативного планирования, применяемог о для данного вида бюджета (см. 
п.2.2.2), будет сформирован перечень используемых а налитик. 
В поле «Использовать для»  пользователь может выбрать область применения 
создаваемой настройки: 
 для всех операций, 
 для лимитов, 
 для резерва, 
 для планирования. 
Вклада «Аналитики»  представляет собой таблицу, в которой перечислены 
аналитики, заданные структурой оперативного планиро вания для выбранного вида 
бюджета (см. Рисунок 11). Для каждой аналитики могу т устанавливаться следующие 
параметры: 
 «Использование»  – указывается вариант отображения аналитики в доку менте 
«Операционный план» . Доступны для выбора значения «в шапке» , «в 
строке» , «в расшифровке» . Для некоторых аналитик значение предустановлено 
настройками Системы и не может быть изменено.  
 «Группа»  – указывается обозначение группы, к которой относи тся аналитика. 
Данный параметр может быть задан только для реквизи тов «Статья бюджета»  и 
«Аналитика1»…«Аналитика6» . 
 «Значение аналитики»  – для каждой аналитики (кроме аналитик «Период» , 
«Вид бюджета»  и «Аналитика1»…«Аналитика6» ) может быть указано её 
значение. Выбор значений осуществляется из соответс твующих справочников. 
 «Обязательно»  – признак, устанавливаемый автоматически, в соотве тствии со 
структурой оперативного планирования для выбранного  вида бюджета. 
 «Контроль лимитов»  – признак, устанавливаемый автоматически, в 
соответствии со структурой оперативного планировани я для выбранного вида 
бюджета.
