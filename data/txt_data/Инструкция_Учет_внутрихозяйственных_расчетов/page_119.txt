  Перенос остатков по 76 -ому счету  между филиалами, Ответственный за 
учет внутрихозяйственных операций  
Обработку «Передача остатков по 76 -ому счету» пользователь может найти 
в подсистеме «Казначейство», раздел «Серви с». 
Форма обработки выглядит следующим образом  (см. рисунок 174 ): 
Рисунок 174  – Форма обработки «Передача остатков по 76 -ому счету»):  
 
 
Пользователь в шапке обработки указывает значения следующих полей  (см. ри-
сунок 175 ): 
 «Период» - дата, на которую анализируются остатки по 76 -ому счету. Данное 
поле обязательно для заполнения.  
 «Организация -отправитель» - организация, по которой анализируются 
остатки по 76 -ому счету. Данное поле обязательно для заполнения;  
 «Счет» - отбор по конкретному субсчету 76 -ого счета. Данное поле опцио-
нально для заполнения.
