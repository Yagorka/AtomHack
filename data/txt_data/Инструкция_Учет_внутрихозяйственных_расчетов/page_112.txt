   Инструкция пользователей «D -1C1 -1.10.50  
Учет внутрихозяйственных расчетов»  
 
Автор  АО «Гринатом»  Инструкция пользователей «D -1C1-1.10.50 
Учет внутрихозяйственных расчетов»  Страница  112 
Страниц  122 
Версия  5 
 
Шапка документа  заполняется следующим образом:  
 «Организация -отправитель » - организация, на которой числятся перенести 
резервы  по зарплате;  
 «Организация -получатель»  - организация, на которую необходимо перене-
сти резервы по зарплате;  
  «Направление деятельности » - отбор по направлению  деятельности, по ко-
торому числятся остатки резервов по зарплате ; 
Табличная часть заполняется путем  нажатия команды « Заполнить по остат-
кам», которая анализирует  остатки по счетам 96.01.1 и 96.01.2 разрезе  субконто. 
Поля в табличной части заполняются следующим образом:  
 «Счет » - счет резервов по зарплате, на котором есть остатки, по умолчанию 
заполняется автоматически ; 
  «Субконто Кт1 » - субконто по счету, на котором есть остатки, по умолча-
нию заполняется автоматически;  
 «Сумма резерва » - сумма резерва на счете, по умолчанию заполняется авто-
матически;   
 «Сумма резерва страховых взносов » - сумма резерва страховых взносов на 
счете, по умолчанию заполняется автоматически;   
 «Сумма резерва ФСС НС и ПЗ » – сумма резерва ФСС НС и ПЗ на счете, по 
умолчанию заполняется автоматически;  
 «Валюта » - указывается валюта, в которой начислены остатки;  
 «Валютная сумма » - указывается валютная сумма, в которой начислены 
остатки;  
 «Направление деятельности » - направление деятельности, на котором  чис-
лятся остатки по зарплате, по умолчанию заполняется автоматически ; 
 «НУ Кт » - указывается сумма передаваемых резервов по зарплате  по НУ;  
 «ПР Кт » - указывается сумма передаваемых резервов по зарплате  по ПР;  
 «ВР Кт » - указывается сумма передаваемых резервов по зарплате по ВР ;
