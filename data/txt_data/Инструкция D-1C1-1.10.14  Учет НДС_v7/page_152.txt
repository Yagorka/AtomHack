По строкам документа «Начисление НДС по СМР», отмеченным флагом 
«Ручная корректировка », автоматической актуализации сумм выполняться не 
будет.  
В отчете по сходимости данных между регистром «Прочие расходы» и 
данными проводок по 08 счету в разрезе субконто «Хозспособ» для таких 
объектов дополнительно будет выведена сумма, зафиксированная с  помощью 
ручной корректировки.  
Для анализа сходимости сумм, учтенных для начисления НДС по СМР 
на основании регистра «Прочие расходы», с суммами в ОСВ по счету 08.03 в 
разрезе субконто «Хозспособ», предусмотрена проверка при помощи отчета 
«Отчет по сходимо сти ОУ и БУ НДС по СМР», который можно сформировать 
после этапа отражения в регламентированном учете.
