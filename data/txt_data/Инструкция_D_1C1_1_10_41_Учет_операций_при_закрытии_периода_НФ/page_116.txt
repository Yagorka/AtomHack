В программе предусмотрена  возможность установить  дату запрета.  Для 
этого  необходимо заполнить «Дату запрета» и с помощью переключателя  
необходимо выбрать  советующее значение:  
 «Для  всех  пользователей»  - при установке  данного  переключателя  запрет  
на редактирование  устанавливается для всех  пользователей  по всем  
организациям;  
 «По пользователям» - данный переключатель следует выбирать, если дату 
запрета  необходимо установить для  конкретного пользователя или группы 
пользователей  (для конкретной  информационной  базы),  или разным  
пользователям  (информационным  базам ) необходимо  установить разные  
значения  даты  запрета.  
При выборе переключателя «По пользователям» становится доступным 
список, в  который  можно  добавлять  пользователей/группы  пользователей  
(информационные  базы) и настраивать для каждого из них свое значение даты 
запрета. Добавить в список нужных пол ьзователей/группу пользователей 
(инфор мационные  базы)  можно по кнопке «Подобрать».  Дата запрета 
устанавливаетс я для каждого пользователя/группы пользователей отдельно
